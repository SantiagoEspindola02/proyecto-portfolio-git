Conocimientos

HTML

Dentro del lenguaje HTML se cual es el uso correcto de cada etiqueta y también en que orden ubicarlas para que la página web que este desarrollando y diseñando tenga un buen posicionamiento en el SEO y de esta forma llegar al alcance de más personas.

CSS

En el CSS considero que tengo buena imaginación a la hora de diseñar una página web ya que soy detallista y me gusta que los trabajos se vean bien esteticamente y prolijos. Siempre busco que se vea todo claro para que ningún usuario tenga problemas a la hora de navegar en el sitio.

JAVASCRIPT

JavaScript es el próximo lenguaje de programación que voy a aprender en la carrera de Coderhouse. Con este lenguaje voy a poder mediante una secuencia de comandos, controlar multimedia, animar imágenes y hacer que sean funcionales otras características específicas de la página web.

REACT

React es el otro curso con el que voy a finalizar la carrera en la plataforma virtual de Coderhouse. En este curso voy a aprender a crear interfaces de usuario que lo que hacen es facilitar el desarrollo de aplicaciones en una sola página. La idea es construir aplicaciones que usan datos que cambian constantemente.
